# Simulação de Máquina de Turing

## 📚 Descrição
Projeto acadêmico desenvolvido para demonstrar o funcionamento de uma Máquina de Turing utilizando Python.

---

## 🎯 Objetivo
Simular o comportamento de uma Máquina de Turing para reconhecimento e processamento de cadeias binárias.

---

## 🚀 Tecnologias Utilizadas
- Python
- Algoritmos
- Estruturas de Dados

---

## ⚙️ Funcionalidades
✔️ Leitura de fita  
✔️ Transição entre estados  
✔️ Simulação de processamento  
✔️ Validação de entradas  

---

## 📖 Conceitos Aplicados
- Computabilidade
- Linguagens Formais
- Máquinas de Estados
- Algoritmos
